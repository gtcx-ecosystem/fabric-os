# sessions

Operational continuity for agent sessions.

- `handoffs/` — durable hand-off docs for the next agent (Protocol 49)
- `ci/` — session-specific CI witnesses (closure bar, open-items, hand-off)
- `transcripts/` — transcript index / pointer to the actual transcript SoR
- `retrospectives/` — per-session or per-story retrospectives
- `archive/` — superseded hand-offs, transcripts, retrospectives
