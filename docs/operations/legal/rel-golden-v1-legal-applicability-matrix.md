# REL-GOLDEN-V1 Legal Applicability Matrix

## Boundary

This matrix records legal work required for `REL-GOLDEN-V1`. It is not legal
advice, agreement execution, signatory approval, or public-claims approval.

## Applicability

| Domain                | Applicability                       | Required artifact                                | Authority                                        |
| --------------------- | ----------------------------------- | ------------------------------------------------ | ------------------------------------------------ |
| NDA / confidentiality | Applicable for pilot                | NDA or explicit waiver                           | Class S                                          |
| MOU / pilot agreement | Applicable for pilot                | MOU, pilot agreement, or equivalent              | Class S                                          |
| Terms of service      | Pending scope                       | terms applicability decision                     | Class S when customer-facing                     |
| Privacy policy        | Pending data scope                  | privacy applicability decision                   | Class S when personal data is involved           |
| DPA / data sharing    | Pending data scope                  | DPA or data-sharing agreement                    | Class S when regulated/personal data is involved |
| IP provenance         | Applicable                          | source/provenance record and IP exception review | Class A/S depending on filing                    |
| Code of conduct       | Applicable                          | policy incorporation record                      | Class A                                          |
| Public claims         | Applicable if external comms issued | legal/privacy/IP claims review                   | Class A                                          |
| Signatory authority   | Applicable                          | named signer and execution evidence              | Class S                                          |

## Required before DSLC ready

- Signed or approved NDA/MOU/pilot agreement.
- Signatory authority evidence.
- Code-of-conduct/policy incorporation approval.
- Legal/privacy/IP claims review for any external or customer-facing claims.

## Required before SHIP ready

- Legal-ops critical controls closed or risk-accepted by authorized human owner.
- Any exceptions registered with owner, expiry, rationale, and approval.
- Public-docs/changelog legal review if pilot materials are externally distributed.

## Current status

`REL-GOLDEN-V1` remains blocked until real Class S legal artifacts and Class A
legal-review approvals are attached to the DSLC/SHIP manifests.
