# Code of Conduct

## Our pledge

We pledge to make participation in this project a professional, harassment-free experience for everyone.

## Our standards

Positive behavior:

- Welcoming and inclusive language.
- Respect for differing viewpoints.
- Graceful acceptance of constructive criticism.
- Focus on what is best for the community and product.

Unacceptable behavior:

- Harassment, trolling, or personal attacks.
- Publishing others' private information without permission.
- Conduct inappropriate in a professional project setting.

## Enforcement

Report concerns to the repository maintainers via the contact in [`SECURITY.md`](./SECURITY.md). Complaints are reviewed promptly and fairly.

## Scope

This Code of Conduct applies in all project spaces and when an individual officially represents the project in public.
