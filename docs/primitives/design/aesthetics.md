# Aesthetics

Visual tone, density, type, color, layout, and imagery.

## Direction

Use this track to record Fabric OS-specific design decisions for the AWS, Kubernetes, Terraform, deployment, and independent assurance control plane. Keep direction practical, reviewable, and grounded in real operator workflows.

## Required Notes

- State what is normative now versus exploratory.
- Link reference material when it changes design direction.
- Avoid decorative patterns that obscure workflow clarity, evidence review, or operational trust.
- Keep implementation details in owner product surfaces, not in this primitive unless they define a reusable rule.
